
Boilerplate code to create reactJS app with `material-ui` and `webpack`. 

Streamlined minimalist build tool-chain, not using `create-react-app`. 

In Box
==
* Material-UI@next
* Webpack 3
* React v16
* React Router
* JS uglifier

Development
==

Install requirements
```bash
npm install
```

Start dev build
```bash
npm start
```

Build production build
```bash
npm run build
```
